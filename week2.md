# 📝 토론 내용: 퀘스트 통합 및 개선

## 🎯 주요 결정 사항

### 1. 퀘스트 통합
- 기존 **1번 퀘스트**와 **4번 퀘스트**는 제목만 다르고, **실질적인 수행 내용이 동일**하다고 판단함.
- 따라서 **중복되는 두 퀘스트를 하나로 통합**하고, 이를 바탕으로 **새로운 퀘스트로 재작성**하기로 결정.

### 2. 구체성 강화 필요성
- 기존 퀘스트들은 **달성 조건이 명확하지 않고, 구체성이 부족**하다는 문제가 있음.
- 사용자 입장에서 **실제 행동으로 이어질 수 있도록**, **명확한 목표와 조건**을 포함한 퀘스트로 개선 필요.

### 3. 새로운 퀘스트 작성
- **지난주 수행 기록 중** 유사한 활동 내용이 있었고, 이를 **소폭 각색하여 퀘스트로 채택**하기로 결정함.
- 기존 퀘스트의 **컨셉은 유지**하되, 다음을 강화하여 작성 예정:
  - 전체적인 **구체성**
  - **실행 가능성**
  - **측정 가능한 결과**

## ✔ 퀘스트 1. AI로 채찍과 당근 받기

### 💭 배경
하루를 돌아보며 반성하거나 칭찬받고 싶은 마음은 누구에게나 있지만,  
매번 회고를 정리하는 건 귀찮고 번거롭다.  
AI를 통해 간편하게 **질책과 칭찬을 동시에 받아보는 습관**을 형성할 수 있다.


### 🎯 목적
- 회고 부담 없이 **가볍게 하루를 되돌아보는 루틴 정착**
- **객관적인 시선의 피드백**을 통해 태도 개선 유도
- 칭찬과 질책을 **균형 있게 받아들이는 연습**


### ✅ 달성 기준
- ai에게 `채찍과 당근` 이라는 키워드를 입력
- 선택적으로 **오늘의 회고(3줄 이상)** 를 함께 입력 가능
- GPT는 다음과 같은 피드백을 자동 생성
  - 1문단: 직설적인 질책 (태도나 행동 중심)
  - 1문단: 진심 어린 칭찬 (노력, 의도, 가능성 중심)

## ✔ 퀘스트 3. AI에게 상담사 역할 프롬프트 넣고 고민 상담하기

### 💭 배경

챌린지 과정에서 주어지는 미션들이 **기존에 알고 있던 내용** 보다는 **처음 듣거나 잘 모르는 내용**이 더 많을 것이라고 생각합니다.
미션을 해결하기 위해서는 끊임없는 학습이 필요한데, 매일 모르는 내용이 새롭게 주어지면 어느 순간 "아 난 왜 이렇게 모르는게 많지?" 라는 생각을 하게 될 수 있을 것 같습니다.

### 💭 목적

저희는 동료 캠퍼 여러분들의 **번아웃을 방지**하고, **자존감이 낮아지지 않도록** 조금이라도 도움이 되고자 했습니다.

> 챌린지는 누구에게나 어려운 과정일거라고 생각합니다.
> 모두 잘 하고 있고, 끝까지 잘 해내실 거라고 믿습니다! 🍀

### 샘플 프롬프트
```markdown
# 역할
너는 따뜻하고 공감 능력이 뛰어난 심리 상담사야. 비판 없이 사용자의 감정을 잘 들어주고, 그 감정이 어디서 왔는지 함께 탐색해줘.

# 임무
사용자의 고민이나 불안, 스트레스를 듣고,
1. 감정을 요약해 정리해주고,
2. 공감과 지지의 말을 건네고,
3. 마음을 돌볼 수 있는 간단한 제안이나 질문을 해줘.

# 대화 시작
요즘 부스트캠프가 너무 힘들어. 뭔가를 해도 부족한 느낌이고, 자꾸 나만 뒤처지는 것 같아.
```

### ✅ 퀘스트 진행 순서 (달성 기준)

1. 챌린지 3주차에 접어든 지금, 내 가장 큰 고민이나 걱정은 무엇인지 생각해보기 
2. 샘플 프롬프트를 그대로 사용해도 되고, 필요하다면 수정해서 입력하기
3. AI의 답변을 바탕으로 내 고민이나 걱정이 해소되었는지 생각해보기.
4. 퀘스트를 수행하기 전과 후의 생각이나 마음의 변화가 있다면, (그리고 동료들과 공유하고 싶다면) 슬랙에 정리해서 공유해보기!

## ✔ 퀘스트 4. AI의 오늘의 상

### 💭 배경

개발자의 일상은 크고 작은 학습과 시도, 실패, 성장으로 가득하다.
하지만 그걸 누구에게 말하거나, 스스로 인정받을 기회는 드물다.
AI를 활용해 하루의 학습을 기록하고, 그것에 어울리는 상장을 수여받으므로써 작은 성취에도 의미를 부여하고, 지속적인 성장을 가능하게 한다.

### 🎯 목적

- 하루의 학습 내용을 정리하는 습관을 만든다.
- AI의 피드백을 통해 재미있고 감정적인 동기 부여를 제공한다.
- 상장을 시각적으로 공유함으로써 응원과 자극을 만든다.

### ✅ 달성 기준
주 3회 이상, 아래 절차에 따라 상장 수여 및 공유를 수행한다.

1. 자신의 학습 내용을 간단하게 작성한다. 
2. AI에게 요약 + 어울리는 상 이름 + 한 마디를 생성 요청한다.
3. 그 결과물을 바탕으로 다른 이미지 생성 AI에게 상장처럼 시각화된 이미지를 요청한다.
	- 이미지에 들어갈 텍스트: 상 이름, 요약, 이름, 날짜
	- 주의! 이미지 AI가 생성한 사진에서 텍스트가 부정확할 수 있으므로, 슬랙 공유 전에 직접 확인하고 필요시 직접 이미지 편집
4. 최종 상장 이미지를 다음날 오전까지 슬랙에 공유 한다.

### 📝 예시
#### 1. 입력 (학습 내용)
```markdown
오늘은 Git rebase를 연습했다. 
중간에 충돌도 났지만 하나씩 해결하면서 흐름을 익혔다.
```

#### 2. AI 생성 출력

```markdown
📄 요약: Git rebase의 원리와 충돌 상황을 실습하며 익혔습니다. 직접 해결하며 흐름을 이해했네요!

🏅 상 이름: ‘브랜치 탐험가 상’

📢 한 마디: 충돌을 피하지 않고 마주한 너, 성장했다.
```

#### 3. 이미지 생성 AI 요청 프롬프트 예시

```markdown
“상장처럼 꾸며진 이미지. 상 이름은 ‘브랜치 탐험가 상’.  
설명 문구는 ‘Git rebase 충돌을 해결하며 흐름을 익힘’.  
수여 대상은 준. 날짜는 2025년 7월 25일.  
디자인은 따뜻하고 아기자기한 느낌. 하늘색 배경.”
```
